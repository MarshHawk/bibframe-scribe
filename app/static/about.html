About the BIBFRAME Scribe prototype

Overview

The Scribe prototype is a tool developed by Zepheira that demonstrates how to catalog various materials natively in Linked Data. Initially supported by the Library of Congress, Scribe has matured with additional support from UC Davis University Library as part of the BIBFLOW project funded by the Institute for Museum and Library Services. Feedback from Zepheira's Linked Data and BIBFRAME Practical Practitioner Training Alumni have also been invaluable in Scribe's continued development.

This prototype is designed to demonstrate how one would catalog entities using the Bibframe Lite vocabulary. Bibframe Lite is an RDF vocabulary designed to integrate with and engage in the wider information community while also serving the very specific needs of local implementation. Zepheira works with a range of communities to extend the Bibframe Lite vocabulary. These additional vocabulary layers include:

  • BF Lite + Library
  • BF Lite + Archives
  • BF Lite + Rare Materials
  • BF Lite + Relation

Prototype Objectives

  • Demonstrate how components of the Bibframe Lite framework come together in an editing platform.
  • Provide a method to examine workflows for cataloging in native Bibframe Lite.
  • Explore common relationships between Works such as translations, derivative works, etc.
  • Demonstrate how different vocabularies are accommodated in a single interface.
  • Explore the ability to link to external services that are needed in workflows, such as VIAF, Library of Congress.
  • Demonstrate unique data entry features such as “look ahead” searches of remote services and a drag and drop image zone for images.
  • Show possible future functionality such as user activity dashboard via mock-up pages.

Prototype Functionality

  • Scribe uses 32 different profiles to demonstrate the flexibility of the framework, including 12 modular micro-profiles for Events, Categories and Concepts. Scribe also includes profiles for electionic and print versions of followingm and all of thish containue default values which can be edited by the cataloger: :
    • Bibliography
    • Book
    • Dissertation
    • Article
    • Vinyl record
    • Map
    • Serial
  • Interface reflects a subset of the BIBFRAME Lite + Library vocabularies. The interface guides you to the particular instance and brings up only fields that are pertinent to that instance type.
  • Search allows users to find local material and select for additional curation and editing.
  • Uses remote search services:
    • Library of Congress Linked Data Service (names, subjects, languages, places, RDA categories)
    • assignFAST (subjects)
    • VIAF (names, subjects)
    • AGROVOC (subjects)
    • Medical Subject Headings (MeSH) RDF Linked Data (subjects)
  • Support identifier disambiguation of subjects, people, and other resources by providing access to data from remote search service in Resource Description Framework (RDF) format. For services that support human readable content negotiation, users can click a link to open the RDF displayed in a new browser tab for context.
